# Linguistic Pseudo Class Selectors

## CSS 2.1 Linguistoc PC Selectors
```
html[lang|="en"] 
a[attr|=val] an en-us en-uk
p:lang(en)
```

## CSS Selectors Level 4
```
:lang(*-ch)
```
